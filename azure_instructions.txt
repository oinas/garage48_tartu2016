Hardened MongoDB v2.2.3: Cognosys brings you MongoDB v2.2.3 on Hardened Ubuntu 12.04 LTS kernel. MongoDB is agile, scalable, high-performance, open source NoSQL database. MongoDB configured is v2.2.3. To use the server open port 27017 and 28017 in Azure load balancer and check your MongoDB server externally using [Youname].cloudapp.net:28017, you would have to create data directory before using, either create default sudo mkdir -p /data/db/ ,sudo chown `id -u` /data/db or assign custom directory with --dbpath For more information please visit http://www.cogno-sys.com/azure-cloud-management/mongodb-azure/
